# Shape-Guided (route-shape-align)

**方形＝四條直線**：規定路段站序均分到正方四邊（只 H/V）。

**硬規則**：交付結果的不当交叉與點重疊不得差於輸入。只釘 W、其餘不動會撕網 → §6 管線為：
1. 整網軸向仿射使 W 外接成正方（保平面）
2. 同框把 W 釘到四邊
3. 其餘站從連續解重鋪＋積極修復
4. 無法同時成方且保規則 → **退回輸入**（不交付蜘蛛網）

**兩處入口**：`layout-shape`（初步直線化⑨）／`hc-<kind>-shape`（循環後）

**規定表**：東京 JR 山手線、新加坡 Circle、東京大江戶環段；其餘不需計算。

`HC_LS_KEY` v27+。
